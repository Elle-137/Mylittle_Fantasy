学习 Web 前端交互式设计是个很棒的方向！WebGL 及其现代封装库（如 Three.js）能让你在浏览器中实现电影级的实时 3D 效果。我来为你梳理一条系统的学习路径，并展示一些可以直接在浏览器中运行的交互式 Demo，帮助你直观理解这些技术的应用场景。

---

## 一、核心技术栈概览

| 层级       | 技术                         | 用途                          | 学习优先级        |
| ---------- | ---------------------------- | ----------------------------- | ----------------- |
| **基础层** | Canvas 2D API                | 2D 绘图、粒子系统、数据可视化 | ⭐⭐⭐ 先掌握        |
| **核心层** | WebGL (原生)                 | 底层 3D 渲染，直接操作 GPU    | ⭐⭐ 理解原理       |
| **框架层** | **Three.js**                 | 最流行的 WebGL 封装库         | ⭐⭐⭐ 重点学习      |
| **进阶层** | GLSL 着色器                  | 自定义顶点/片元着色器         | ⭐⭐ 后期深入       |
| **特效层** | GSAP / Lottie                | 交互动画编排、复杂时间线      | ⭐⭐⭐ 配合 Three.js |
| **性能层** | OffscreenCanvas / Web Worker | 多线程渲染，避免主线程阻塞    | ⭐ 进阶优化        |

## 二、学习路径建议

### Phase 1: 夯实基础（1-2 周）
1. **Canvas 2D 基础**：路径绘制、渐变、合成模式、像素操作
2. **数学基础**：向量、矩阵变换（旋转/缩放/平移）、三角函数
3. **动画原理**：`requestAnimationFrame` 循环、时间差分（delta time）、缓动函数（Easing）

### Phase 2: WebGL 核心概念（2-3 周）
1. **渲染管线**：顶点着色器 → 图元装配 → 光栅化 → 片元着色器
2. **核心对象**：Buffer、Attribute、Uniform、Texture、FrameBuffer
3. **坐标系统**：模型空间 → 世界空间 → 观察空间 → 裁剪空间 → 屏幕空间
4. **光照模型**：Lambert（漫反射）、Phong/Blinn-Phong（镜面反射）

### Phase 3: Three.js 实战（3-4 周）
1. **场景图**：Scene → Camera → Renderer → Mesh → Geometry + Material
2. **几何体**：BufferGeometry、自定义顶点数据、InstancedMesh（实例化渲染）
3. **材质系统**：ShaderMaterial（自定义 GLSL）、RawShaderMaterial
4. **后期处理**：EffectComposer、Bloom、SSAO、FXAA 等后处理通道
5. **交互集成**：OrbitControls、Raycaster（鼠标拾取）、与 DOM 事件结合

### Phase 4: 高级特效与性能（持续）
1. **自定义着色器**：噪声函数（Simplex/Perlin）、Ray Marching、SDF（有向距离场）
2. **物理模拟**：粒子系统（GPU 粒子）、流体模拟、布料模拟
3. **性能优化**：Frustum Culling、LOD、纹理压缩（KTX2/Basis）、Draw Call 合并

// ======== PARTICLE SYSTEM ========
class ParticleSystem {
  constructor(count) {
    this.particles = [];
    for (let i = 0; i < count; i++) {
      this.particles.push({
        x: Math.random() * width,
        y: Math.random() * height,
        vx: (Math.random() - 0.5) * 2,
        vy: (Math.random() - 0.5) * 2,
        radius: Math.random() * 2 + 1,
        hue: Math.random() * 60 + 240
      });
    }
  }

  update(dt) {
    this.particles.forEach(p => {
      const dx = mouse.x - p.x;
      const dy = mouse.y - p.y;
      const dist = Math.sqrt(dx*dx + dy*dy);
      
      if (dist < 300 && dist > 1) {
        const force = mouse.isDown ? -800 : 300;
        const f = force / (dist * dist) * 100;
        p.vx += (dx / dist) * f * dt;
        p.vy += (dy / dist) * f * dt;
      }
      
      // Friction
      p.vx *= 0.96;
      p.vy *= 0.96;
      
      // Update position
      p.x += p.vx;
      p.y += p.vy;
      
      // Boundary bounce
      if (p.x < 0 || p.x > width) { p.vx *= -0.8; p.x = Math.max(0, Math.min(width, p.x)); }
      if (p.y < 0 || p.y > height) { p.vy *= -0.8; p.y = Math.max(0, Math.min(height, p.y)); }
    });
  }

  draw(ctx) {
    ctx.fillStyle = 'rgba(10, 10, 15, 0.2)';
    ctx.fillRect(0, 0, width, height);
    
    this.particles.forEach(p => {
      const speed = Math.sqrt(p.vx*p.vx + p.vy*p.vy);
      const alpha = Math.min(1, speed / 5 + 0.3);
      const size = p.radius + speed * 0.3;
      
      ctx.beginPath();
      ctx.arc(p.x, p.y, size, 0, Math.PI * 2);
      ctx.fillStyle = `hsla(${p.hue}, 80%, 60%, ${alpha})`;
      ctx.fill();
      
      // Glow effect
      ctx.beginPath();
      ctx.arc(p.x, p.y, size * 3, 0, Math.PI * 2);
      ctx.fillStyle = `hsla(${p.hue}, 80%, 60%, ${alpha * 0.1})`;
      ctx.fill();
    });
  }
}

// ======== WAVE GRID ========
class WaveGrid {
  constructor() {
    this.cols = 30;
    this.rows = 20;
    this.time = 0;
  }

  update(dt) {
    this.time += dt * 2;
  }

  draw(ctx) {
    ctx.fillStyle = 'rgba(10, 10, 15, 0.3)';
    ctx.fillRect(0, 0, width, height);
    
    const cellW = width / this.cols;
    const cellH = height / this.rows;
    
    for (let i = 0; i <= this.cols; i++) {
      for (let j = 0; j <= this.rows; j++) {
        const x = i * cellW;
        const y = j * cellH;
        
        // Distance from mouse
        const dx = mouse.x - x;
        const dy = mouse.y - y;
        const dist = Math.sqrt(dx*dx + dy*dy);
        const mouseInfluence = Math.max(0, 1 - dist / 200) * 30;
        
        // Sine wave height
        const wave = Math.sin(i * 0.3 + this.time) * Math.cos(j * 0.3 + this.time * 0.7) * 15;
        const height = 5 + Math.abs(wave) + mouseInfluence;
        
        const hue = (i * 5 + j * 3 + this.time * 20) % 360;
        
        ctx.beginPath();
        ctx.arc(x, y, height * 0.4, 0, Math.PI * 2);
        ctx.fillStyle = `hsla(${hue}, 70%, 60%, 0.8)`;
        ctx.fill();
      }
    }
    
    // Draw connections
    ctx.strokeStyle = 'rgba(99, 102, 241, 0.15)';
    ctx.lineWidth = 1;
    for (let i = 0; i < this.cols; i++) {
      ctx.beginPath();
      for (let j = 0; j <= this.rows; j++) {
        const x = i * cellW;
        const y = j * cellH;
        const wave = Math.sin(i * 0.3 + this.time) * Math.cos(j * 0.3 + this.time * 0.7) * 15;
        const dx = mouse.x - x;
        const dy = mouse.y - y;
        const dist = Math.sqrt(dx*dx + dy*dy);
        const mouseInfluence = Math.max(0, 1 - dist / 200) * 30;
        const offsetY = wave + mouseInfluence;
        
        if (j === 0) ctx.moveTo(x, y + offsetY);
        else ctx.lineTo(x, y + offsetY);
      }
      ctx.stroke();
    }
  }
}

// ======== CONSTELLATION ========
class Constellation {
  constructor() {
    this.points = [];
    this.connections = [];
    for (let i = 0; i < 60; i++) {
      this.points.push({
        x: Math.random() * width,
        y: Math.random() * height,
        vx: (Math.random() - 0.5) * 0.5,
        vy: (Math.random() - 0.5) * 0.5,
        radius: Math.random() * 2 + 1
      });
    }
  }

  update(dt) {
    this.points.forEach(p => {
      p.x += p.vx;
      p.y += p.vy;
      
      if (p.x < 0 || p.x > width) p.vx *= -1;
      if (p.y < 0 || p.y > height) p.vy *= -1;
      
      // Mouse attraction
      const dx = mouse.x - p.x;
      const dy = mouse.y - p.y;
      const dist = Math.sqrt(dx*dx + dy*dy);
      if (dist < 150) {
        p.vx += dx * 0.001;
        p.vy += dy * 0.001;
      }
    });
  }

  draw(ctx) {
    ctx.fillStyle = 'rgba(10, 10, 15, 0.2)';
    ctx.fillRect(0, 0, width, height);
    
    // Draw connections
    ctx.lineWidth = 1;
    for (let i = 0; i < this.points.length; i++) {
      for (let j = i + 1; j < this.points.length; j++) {
        const dx = this.points[i].x - this.points[j].x;
        const dy = this.points[i].y - this.points[j].y;
        const dist = Math.sqrt(dx*dx + dy*dy);
        
        if (dist < 120) {
          const alpha = (1 - dist / 120) * 0.5;
          ctx.strokeStyle = `rgba(129, 140, 248, ${alpha})`;
          ctx.beginPath();
          ctx.moveTo(this.points[i].x, this.points[i].y);
          ctx.lineTo(this.points[j].x, this.points[j].y);
          ctx.stroke();
        }
      }
    }
    
    // Draw points
    this.points.forEach(p => {
      ctx.beginPath();
      ctx.arc(p.x, p.y, p.radius, 0, Math.PI * 2);
      ctx.fillStyle = '#818cf8';
      ctx.fill();
      
      ctx.beginPath();
      ctx.arc(p.x, p.y, p.radius * 4, 0, Math.PI * 2);
      ctx.fillStyle = 'rgba(129, 140, 248, 0.1)';
      ctx.fill();
    });
    
    // Mouse glow
    ctx.beginPath();
    ctx.arc(mouse.x, mouse.y, 20, 0, Math.PI * 2);
    ctx.fillStyle = 'rgba(99, 102, 241, 0.2)';
    ctx.fill();
  }
}

// ======== RIPPLE SYSTEM ========
class RippleSystem {
  constructor() {
    this.ripples = [];
    this.time = 0;
    this.autoRippleTimer = 0;
  }

  update(dt) {
    this.time += dt;
    this.autoRippleTimer += dt;
    
    // Auto generate ripple
    if (this.autoRippleTimer > 2) {
      this.ripples.push({
        x: Math.random() * width,
        y: Math.random() * height,
        radius: 0,
        maxRadius: 150 + Math.random() * 100,
        alpha: 1,
        speed: 80 + Math.random() * 40
      });
      this.autoRippleTimer = 0;
    }
    
    // Mouse click ripple
    if (mouse.isDown && !this.lastMouseDown) {
      this.ripples.push({
        x: mouse.x,
        y: mouse.y,
        radius: 0,
        maxRadius: 200,
        alpha: 1,
        speed: 100
      });
    }
    this.lastMouseDown = mouse.isDown;
    
    // Update ripples
    this.ripples = this.ripples.filter(r => r.alpha > 0.01);
    this.ripples.forEach(r => {
      r.radius += r.speed * dt;
      r.alpha = 1 - (r.radius / r.maxRadius);
    });
  }

  draw(ctx) {
    ctx.fillStyle = 'rgba(10, 10, 15, 0.15)';
    ctx.fillRect(0, 0, width, height);
    
    this.ripples.forEach(r => {
      // Outer ring
      ctx.beginPath();
      ctx.arc(r.x, r.y, r.radius, 0, Math.PI * 2);
      ctx.strokeStyle = `rgba(99, 102, 241, ${r.alpha * 0.8})`;
      ctx.lineWidth = 3;
      ctx.stroke();
      
      // Inner glow
      ctx.beginPath();
      ctx.arc(r.x, r.y, r.radius * 0.7, 0, Math.PI * 2);
      ctx.strokeStyle = `rgba(192, 132, 252, ${r.alpha * 0.4})`;
      ctx.lineWidth = 1;
      ctx.stroke();
      
      // Center dot
      if (r.radius < 20) {
        ctx.beginPath();
        ctx.arc(r.x, r.y, 5 - r.radius * 0.2, 0, Math.PI * 2);
        ctx.fillStyle = `rgba(255, 255, 255, ${r.alpha})`;
        ctx.fill();
      }
    });
    
    // Grid background
    ctx.strokeStyle = 'rgba(99, 102, 241, 0.05)';
    ctx.lineWidth = 1;
    const gridSize = 40;
    for (let x = 0; x < width; x += gridSize) {
      ctx.beginPath();
      ctx.moveTo(x, 0);
      ctx.lineTo(x, height);
      ctx.stroke();
    }
    for (let y = 0; y < height; y += gridSize) {
      ctx.beginPath();
      ctx.moveTo(0, y);
      ctx.lineTo(width, y);
      ctx.stroke();
    }
  }
}

let particles = new ParticleSystem(800);
let waves = new WaveGrid();
let constellation = new Constellation();
let ripples = new RippleSystem();

function animate(timestamp) {
  const dt = Math.min((timestamp - lastTime) / 1000, 0.1);
  lastTime = timestamp;

  ctx.clearRect(0, 0, width, height);

  switch(currentTab) {
    case 'particles':
      particles.update(dt);
      particles.draw(ctx);
      stats.textContent = `FPS: ${Math.round(1/dt)} | Particles: 800 | Gravity: ${mouse.isDown ? 'Repel' : 'Attract'}`;
      break;
    case 'waves':
      waves.update(dt);
      waves.draw(ctx);
      stats.textContent = `FPS: ${Math.round(1/dt)} | Grid: 30x20 | Time: ${waves.time.toFixed(1)}`;
      break;
    case 'constellation':
      constellation.update(dt);
      constellation.draw(ctx);
      stats.textContent = `FPS: ${Math.round(1/dt)} | Nodes: 60 | Connections: Dynamic`;
      break;
    case 'ripple':
      ripples.update(dt);
      ripples.draw(ctx);
      stats.textContent = `FPS: ${Math.round(1/dt)} | Ripples: ${ripples.ripples.length} | Click to spawn`;
      break;
  }

  animationId = requestAnimationFrame(animate);
}

## 四、Three.js 入门实战：旋转立方体

当你掌握了 Canvas 2D 的基础后，下一步就是进入 Three.js 的世界。以下是一个最基础的 Three.js 场景代码，你可以直接在 HTML 文件中运行：

```html
<!DOCTYPE html>
<html>
<head>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>
  <style>body { margin: 0; overflow: hidden; background: #000; }</style>
</head>
<body>
<script>
  // 1. 创建场景、相机、渲染器（Three.js 三大件）
  const scene = new THREE.Scene();
  const camera = new THREE.PerspectiveCamera(75, window.innerWidth/window.innerHeight, 0.1, 1000);
  const renderer = new THREE.WebGLRenderer({ antialias: true });
  renderer.setSize(window.innerWidth, window.innerHeight);
  document.body.appendChild(renderer.domElement);

  // 2. 创建几何体 + 材质 = 网格
  const geometry = new THREE.BoxGeometry(2, 2, 2);
  const material = new THREE.MeshStandardMaterial({ 
    color: 0x6366f1,
    roughness: 0.3,
    metalness: 0.7
  });
  const cube = new THREE.Mesh(geometry, material);
  scene.add(cube);

  // 3. 添加光照
  const light = new THREE.DirectionalLight(0xffffff, 1);
  light.position.set(5, 5, 5);
  scene.add(light);
  scene.add(new THREE.AmbientLight(0x404040));

  // 4. 相机位置
  camera.position.z = 5;

  // 5. 动画循环
  function animate() {
    requestAnimationFrame(animate);
    cube.rotation.x += 0.01;
    cube.rotation.y += 0.01;
    renderer.render(scene, camera);
  }
  animate();

  // 6. 响应式
  window.addEventListener('resize', () => {
    camera.aspect = window.innerWidth / window.innerHeight;
    camera.updateProjectionMatrix();
    renderer.setSize(window.innerWidth, window.innerHeight);
  });
</script>
</body>
</html>
```
